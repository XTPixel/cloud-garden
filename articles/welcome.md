# 欢迎来到我的个人小站

## 怎么就搞了这么个网站

但很长一段时间，这个想法就只是想法。

直到偶然看到 lvy 大佬的网站和项目经历，真的被震撼到了。[链接在此](https://lvyovo-wiki.tech/)，膜拜大佬。于是又把念头拾了起来，请教了 Codex 和 Claude 两位老师（Claude 老师，我可还记得你~），很快搭出了基础功能。

所以就有了这个叫 **云外庭院** 的小地方。名字嘛，就是想要那种“世界之外、安安静静”的感觉，能在繁杂的生活里寻到一片自己的理想乡。

## 现在都有啥了

东西还挺多的，我一个一个说。

### 首页——灵动的卡片布局

打开首页就是一排可以拖来拖去的小卡片：

- **旅人档案**：头像、名字、导航菜单，长得像个游戏角色面板。
- **云端浮岛**：纯 CSS 画的小岛在飘，打开页面就安静下来了。
- **每日问候**：根据早中晚说不同的话，旁边还有一只 `•ᴗ•` 小精灵会眨眼睛。
- **电子钟**：时钟本钟，另外还塞了番茄钟、正计时、倒计时，上面标签页一点就切。
- **日历**：月历加年度进度条，看一眼就知道今年已经过去多少了——通常看了都沉默。
- **个人链接**：欢迎访问我的 GitHub、Bilibili、小红书（虽然现在啥也没有），我准备转生成为内容创作者。
- **箴言**：点一下换一句随机箴言，都是我很喜欢的句子。
- **音乐播放器**：目前只有一首《晚安喵》，够循环了。
- **天气**：靠浏览器定位获取，至少知道今天是热还是冷。
- **桌面便签**：双击就能写，随手记灵感，关掉也还在。

所有卡片都能**自由拖动**，位置会自动存下来，刷新不丢。窗口拉大缩小时它们也会自动挪位置，不会叠在一起。

哦对，右上角可以**切日间/夜间主题**，切完后所有卡片都会平滑变色。鼠标移动还有**小光点拖尾**，第一次实现的时候，我自己盯着晃了好一会儿。

本来还想做一个**罗小黑电子宠物**，能在网页里乱跑，但实现效果不太满意，先放弃了（有待后续开发）。

### 子页面

除了首页，还有几个独立页面：

- **开发项目**：从 GitHub 拉了仓库列表过来展示。
- **近期文章**：就是你现在看的这个系统，本地 Markdown 文件管理，按时间排序。
- **照片墙**：还在慢慢整理照片。
- **网站收藏**：收集了一些好用的工具和有意思的站点。

## 是用什么做的

说出来你可能不信，**一个框架都没用**（至少现在没有，之后应该会学习相关框架）

全部是原生 HTML + CSS + JavaScript（ES Modules 那种）。没有 React、没有 Vue、没有构建工具、没有 npm install 跑半天。就是打开编辑器写代码，保存，刷新浏览器看效果。

技术选型上，主打一个“能跑就行”：

- **样式**：纯 CSS，靠自定义属性切主题，玻璃质感卡片
- **脚本**：原生 JS，按功能拆成 modules 和 pages
- **数据**：丢 localStorage
- **文章**：自己写了个简易 Markdown 解析器，渲染 `.md` 文件
- **天气**：浏览器定位 + 免费天气 API
- **项目列表**：直接调 GitHub REST API
- **部署**：纯静态文件，丢 GitHub Pages 就行
